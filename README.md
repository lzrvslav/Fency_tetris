# Fency Tettris 🎮

A modern, colorful take on classic Tetris — built with HTML, CSS, and JavaScript.

## Features

- Intuitive keyboard controls
- Clean, responsive UI
- Live timer + score counter
- Animated scoring effects
- Pause menu (Continue / Restart / Quit)
- Background music

## Controls

- ⬅️ / ➡️ — Move
- ⬇️ — Soft drop
- ⬆️ — Rotate
- Esc — Pause menu
